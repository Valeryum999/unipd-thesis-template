# Struttura del template

Il template è strutturato nel seguente modo:
```
files
├── appendix/
│   ├── bibliography.bib
│   ├── bibliography.tex
│   └── glossary-entries.tex
|
├── chapters/
│   ├── 1_introduction.tex
│   └── 2_processes.tex
|   └── ...
|
├── config/
│   ├── packages.tex
│   ├── thesis-config.tex
│   └── variables.tex
|
├── images/
│   ├── unipd-logo.png
│   └── ...
|
├── preface/
│   ├── 1_title-page.tex
│   ├── 2_copyright.tex
│   ├── 3_acknowledgements.tex
│   ├── 4_abstract.tex
│   └── 5_table-of-contents.tex
│   
├── latexmkrc
├── output.xmpdata
└── thesis.tex
```
